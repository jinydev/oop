객체 복제 호출

__clone() 매직 메소드는 객체가 복제 되었을 때 실행회는 메서드입니다.

예제파일) class-17.php

<?php

    class MyClass
    {
        public $instance;

        public function __clone() {
            echo "clone object<br>";
        }

    }
        
    $obj = new MyClass();

    $obj2 = clone $obj;

?>
