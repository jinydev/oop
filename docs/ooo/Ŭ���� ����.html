클래스 계층

관점

클래스 계층에서 대부분시 관점의 포인트는 하위 클래스 부터 생각을 하게 됩니다.

하위 클래스 부터 관점을 가지는 이유는 상위 클래스의 정의된 메소드를 하위 클래스에서 사용을 하기 때문입니다.

또한, 하위 클래스는 상위클래스에 없는 새로운 매소드를 추가할 수 있기 때문입니다.

만일 상위 클래스의 매소드가 존재하는 경우 오버라이드를 통하여 이를 변경할 수 있기 때문입니다.
